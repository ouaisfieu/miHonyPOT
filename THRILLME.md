# Investigation du site miHonyPOT : un outil éducatif belge, pas une arnaque

Ce site n'est **pas un phishing malveillant** mais fait partie d'un projet belge de sensibilisation citoyenne. Le nom "miHonyPOT" est un jeu de mots délibéré sur "honeypot" (pot de miel en cybersécurité), suggérant une vocation pédagogique de démonstration plutôt qu'une intention frauduleuse. Le contenu exploite la controverse autour du documentaire de Christophe Deborsu sur la fraude sociale pour sensibiliser aux risques de manipulation.

## Le compte GitHub "ouaisfieu" : un projet citoyen belge

L'opérateur du site est le collectif **ouaisfi.eu**, un projet de "veille citoyenne artisanale" belge focalisé sur l'éducation populaire et l'analyse critique. Le compte GitHub héberge plusieurs sous-sites :

- **DOCTech** : notes de recherche sur la politique belge
- **BXL2030** : analyses prospectives sur Bruxelles  
- **Dossiers** : études critiques sur l'économie, la citoyenneté et l'information
- **GPTPardi** : contenu de recherche assistée par IA

Le projet affiche une transparence assumée : code source public, licence CC BY-SA 4.0, pas de publicité ni de tracking, et contact identifiable (souad.effek@ouaisfi.eu, @ouaisfi.eu sur Bluesky). Le collectif travaille vers une reconnaissance officielle par le **Décret Éducation permanente** de la Fédération Wallonie-Bruxelles d'ici mars 2026.

## Pourquoi "miHonyPOT" ? Un outil de démonstration éducative

Le terme combine plusieurs références : "mi" (demi en français), "Hony" (déformation de "Honey"), et "POT" en majuscules pour souligner le concept de "honeypot". En cybersécurité, un honeypot est un **leurre conçu pour détecter ou démontrer des attaques**. La structure URL `app/deborsu/fraude-sociale` suggère fortement qu'il s'agit d'une simulation pédagogique montrant comment fonctionnent les pages de phishing liées aux prestations sociales.

Cette interprétation est cohérente avec la mission du projet : éduquer les citoyens à reconnaître les faux sites officiels et développer leur esprit critique face aux manipulations numériques.

## Christophe Deborsu : exploitation d'une controverse réelle

**Christophe Deborsu** est un journaliste politique belge légitime travaillant pour RTL Belgium depuis 2015. En **novembre 2025**, son documentaire "Sans boulot : tous fraudeurs ?" diffusé dans son émission "Je vous dérange" a provoqué une tempête médiatique :

- **410 860 téléspectateurs** (39,1% de part de marché)
- Plus de **100 plaintes au CSA** (Conseil Supérieur de l'Audiovisuel)
- Ouverture d'une enquête pour "incitation potentielle à la discrimination sociale"
- Débat politique intense : MR favorable, PS et gauche très critiques

Le site miHonyPOT **n'est absolument pas affilié à Deborsu ni à RTL Belgium**. Il exploite cette controverse très médiatisée pour créer un outil éducatif sur le thème de la fraude sociale — probablement pour montrer comment des acteurs malveillants pourraient usurper ce sujet d'actualité.

## Évaluation des risques : faible dangerosité

**Aucun signalement de phishing ou d'arnaque** n'a été trouvé concernant ce site dans les bases de données de sécurité (VirusTotal, URLScan) ni sur les plateformes de signalement françaises et belges. L'analyse révèle plusieurs indicateurs rassurants :

| Critère | Observation |
|---------|-------------|
| Monétisation | Aucune (pas de pubs, pas de formulaires de paiement) |
| Collecte de données | Le site affirme ne pas tracker les visiteurs |
| Transparence | Code source public, licence clairement affichée |
| But déclaré | Éducation citoyenne et analyse critique |
| Contexte | Projet académique/citoyen visant reconnaissance officielle |

Cependant, **la prudence reste de mise** : le nom "miHonyPOT" indique que le site contient probablement des **simulations de pages frauduleuses** à des fins pédagogiques. Ne saisissez jamais de vraies informations personnelles, même sur un site éducatif.

## Recommandations pratiques

Ce site apparaît comme un **projet légitime de sensibilisation citoyenne**, pas une arnaque. Néanmoins :

- **Ne fournissez aucune donnée personnelle réelle** — même si c'est éducatif, les simulations de phishing ne nécessitent pas vos vraies informations
- Le contenu officiel de Christophe Deborsu se trouve exclusivement sur **rtl.be et rtlinfo.be**
- En cas de doute sur un site suspect en Belgique, signalez à **suspicious@safeonweb.be**
- Le projet ouaisfi.eu semble transparent dans ses intentions, mais reste un acteur indépendant sans affiliation institutionnelle formelle (pour l'instant)

La conclusion : vous n'avez probablement pas reçu un lien malveillant, mais plutôt un lien vers un outil de démonstration pédagogique créé par des activistes civiques belges. Le caractère provocateur du nom "miHonyPOT" et l'utilisation du nom de Deborsu sont des choix délibérés pour attirer l'attention sur les risques de fraude numérique dans le contexte du débat sur la fraude sociale en Belgique.
